# Functions list

* 複数のビルド設定ファイル対応
* ビルドフラグ分岐対応
  * DevelopmentBuild
  * ClearBuildCache
  * Connect With Profiler
  * Deep Prilling
* ビルド先のフォルダを自動作成機能
  * 日時で作成
  * 前ビルドしたディレクトに上書き
* ビルド対象にするシーンを選択
* ビルド後にビルド先フォルダを開く
* ビルド後に自動実行
* iOS,AndroidでアプリID系の上書き
* オリジナル処理、設定ファイルの編集と追加
* アプリ定義ファイル追加及びビルド時の切り替え追加(ver2.0.0)
